# Tarot Card Images

The 78 card face images in this directory are local 420px-wide JPEG derivatives of Wikimedia Commons files from:

- https://commons.wikimedia.org/wiki/Category:Rider-Waite_tarot_deck_(Roses_%26_Lilies)

This Commons category describes the set as the original "Roses & Lilies" edition of the Rider-Waite-Smith tarot deck published by the Rider Company in 1909.

The project uses local copies so the website works from GitHub Pages without relying on hotlinked external images.
